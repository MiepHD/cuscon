Dieses Icon-Pack mit über 5.000 Icons bringt Abwechslung und Dynamik auf deinen Homescreen. Ohne Hintergrund sieht jedes Icon einzigartig aus und durchbricht das konstante Raster. Alle Icons haben einen schwarzen Rand, sodass sie auf den meisten Hintergründen gut zur Geltung kommen. Die Farbpalette wird beibehalten und in das jeweilige Icon integriert. Dadurch sind die Icons leicht zu erkennen. Cuscon sorgt auch für eine natürliche Konsistenz in der Größe der Icons. Alle Icons haben den gleichen Abstand zueinander, sodass sie eine Standardgröße haben, ohne dass sie maskiert werden müssen. Im Gegensatz zu vielen anderen Icon-Packs unterstützt die Philosophie von Cuscon die Erstellung von Icons für Spiele und andere komplexe Anwendungen. Daher ist es möglich, Cuscon vollständig anzuwenden, ohne dass lästige, unbearbeitete Icons entstehen, die nicht zum Thema passen. Da Cuscon Open Source ist, kann jeder auf GitHub Icons und andere Änderungen beisteuern. Cuscon unterstützt die meisten Launcher, darunter Apex, Nova, Smart, Kiss und Lawnchair. Jeder kann Icons anfordern, die noch nicht zur Anwendung hinzugefügt wurden. Die meisten Kalender können anhand des Datums aktualisiert werden, sofern der richtige Launcher verwendet wird.
